INSTRUCCIONES DE VIUALIZACIÓN  DE LA APLICACIÓN DataMundial

El nombre de la apliación es: DataMundial
Para ejecutarla se debe iniciar en la terminal a través del comando: python manage.py runserver

Una vez ejecutado el comando se habilitará la ventana de Inicio. Desde allí se puede ingresar a las siguientes opciones:
Inicio, Registro, Concurso, Selecciones, Pronosticos

Las opciones poseen la siguiente función:
Inicio:          Redirige la página de Inicio
Registro:        Envía a la página donde se puede hacer el registro de personas y consultar personas ya registradas.
Concurso:        en construcción
Selecciones:     muestra información de las selecciones participantes según datos en la base de datos.
Pronosticos:     en construcción
